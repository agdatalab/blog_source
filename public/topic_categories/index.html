Tool - R, Python

주요 토픽을 다루는 방법과 모델이 구동되는 도구

  1. R - 통계기반 모델링을 위한 최고의 친구
  2. R shiny - R 분석 결과를 웹에서 대시보드 형태로 보고싶을 때
  3. R blogdown - R 분석 결과를 웹에서 블로그 형태로 보고싶을 때
  4. Python - R보다 머신러닝과 웹 연계 기능에서 앞서가는 친구
  5. Keras - Tensorflow를 이용한 딥러닝을 쉽게 할 수 있게 해주는 친구
